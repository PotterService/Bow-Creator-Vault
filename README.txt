Bow Creator Vault - Public Website

Upload these files to GitHub Pages:
- index.html
- style.css
- script.js
- creator-cards.json
- images/ folder

To add real cards, edit creator-cards.json.

Image paths:
If your image is inside the images folder, use:
images/my-card.png

If your image is hosted online, use the full image URL.

Testing locally:
Use VS Code Live Server or run:
python -m http.server 8000
Then open http://localhost:8000

Important:
Opening index.html by double-clicking may block creator-cards.json from loading.

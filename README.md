# Bow Creator Vault

Bow Creator Vault (BCV) is a custom trading card registry and management platform designed to organize, showcase, and preserve creator-owned card projects.

Features include:

* Card registry and database
* Set management
* Booster pack and booster box support
* Card image storage
* Print file management
* Collection tracking
* Public card pages
* Creator-owned card archives

Bow Creator Vault serves as the official home for custom card projects, including Myth & Mischief and future creator card releases.
